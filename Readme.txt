Evil Ras interface
-----------------------
* Used Pure Evil as base code
* Mostly Ras Interface elements
* Pure Evil: Map 
* Pure Evil: Pipboy, Fixboy, and Mapboy
* Pure Evil: Scope / Aimed shot menu

* Ras Interface: Registration (modified)
